# NFT Preview Card Component

This is a solution to the NFT Preview Card Component challenge on Frontend Mentor.

## Overview

A responsive card layout built using HTML and CSS, following the provided design.

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Mobile-first approach

## What I Learned

- Centering layouts using Grid and Flexbox
- Using `max-width` and `width: 100%` for responsive cards
- Creating image overlays with `position: absolute`
- Handling hover states for interactive elements
- Scoping CSS selectors properly
- Using `rem` units for spacing
- Managing layout without fixed widths
